Amb

given two or more source Observables, emit all of the items from only the first of these Observables to emit an item

When you pass a number of source Observables to Amb, it will pass through the emissions and notifications of exactly one of these Observables: the first one that emits an item to Amb. It will ignore and discard the emissions of all of the other source Observables.
